# CaWiLyrics 🎤

A beautiful, immersive full-screen lyrics overlay extension for Spicetify. It replaces or enhances the standard lyrics view with fluid animations and dynamic visual effects.

## ✨ Features
* **Volumetric Wave Visualizer:** Dynamic fluid waves on the sides that react when music is playing.
* **Starfield Effect:** Animated floating stars background that speeds up based on playback.
* **Smooth Scrolling:** Uses GSAP for ultra-smooth lyric transitions and centering.
* **Built-in Mini Player:** Control your music (Play/Pause, Next, Prev, Progress bar) directly from the overlay.
* **Playlist Integration:** Quick access button to add the current track to any of your playlists without closing the lyrics.

## 🛠️ Installation

### Via Spicetify Marketplace (Recommended)
Once approved, just search for `CaWiLyrics` in the Extensions tab of your Spicetify Marketplace and click **Install**.

### Manual Installation
If you want to test it right now:
1. Download `CaWiLyrics.js`.
2. Copy it into your Spicetify `Extensions` folder.
3. Run the following commands in your terminal:
   ```bash
   spicetify config extensions cawilyrics.js
   spicetify apply
